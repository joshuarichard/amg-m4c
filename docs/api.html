get /api/children/:id
Gets a single Child object where { _id: req.params.id }.

Return type:

child

Responses:

200

Successful response
Up 
get /api/unsponsored
Gets Child objects where { status: Waiting for Sponsor - No Prior Sponsor }. Limit 100 objects.

Return type:

child

Responses:

200

Successful response
Up 
post /api/donor
Posts (inserts) donor objects into the donor collection. The document to insert should be the JSON body of the request.

Return type:

result

Responses:

200

Successful response
Up 
put /api/donor
Puts (edits) donor objects in the donor collection. Changes should be formatted { _id: "xxxxxx", changes: { "key": "value" } } in the JSON body of the request.

Return type:

result

Responses:

200

Successful response
